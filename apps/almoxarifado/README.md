# Almoxarifado — Cérebro Rural

> App independente e completo de gestão de almoxarifado rural.
> Controla entrada de mercadoria, saída por setor, quem retirou e pra quê.

---

## Princípios

1. **Funciona 100% sozinho** — não depende de nenhum outro app do Cérebro Rural
2. **Offline-first** — galpão não tem Wi-Fi, sistema funciona assim mesmo
3. **Rastreabilidade total** — todo item tem histórico: quem trouxe, quem pegou, pra onde foi
4. **Decisão fica no app** — o almoxarifado classifica cada saída pelo seu próprio destino (setor). O Cérebro só consolida o que o app já decidiu
5. **Sincronização é o último passo** — quando passa em todos os testes, aí sim vira lançamento no fluxo de caixa central
